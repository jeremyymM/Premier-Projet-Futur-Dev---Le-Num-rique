Les Jeux Vidéos

Un jeu vidéo est un jeu qui nécessite un dispositif informatique tel qu’un ordinateur ou une console de jeu, dans lequel le joueur agit sur un environnement virtuel.

A quoi servent t'ils ?

  • -Divertissement
  • -Jeux éducatifs
  • -Jeux de simulation

Les métiers en lien avec les Jeux Vidéos

  • -Game Designer: Donne la direction d’un projet (Histoire, gameplay) il définira et pilotera toutes les phases de développement du jeu.
  • -Programmeur: Programmation du jeu
  • -Graphiste: Aspect Graphique du jeu (Modèle 3D,2D, ArtWork)
  • -Animateur: Anime les modèles 3D et 2D ainsi que les cinématiques.
  • -Testeur qualité: Test et cherche les éventuels bugs encore présent dans le jeu.
